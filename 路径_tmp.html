路径的写法

站内资源和站外资源

站内资源:当前网站的资源 站外资源：非当前网站的资源

绝对路径和相对路径

站外资源:绝对路径 站内资源:相对路径（从当前位置出发）

  1. 绝对路径书写格式: url地址: 协议名://主机名:端口号/路径 schema://host:port/path
http://acm.scau.edu.cn:8000
http://127.0.0.1:5500/2799.html

协议名:http、https、file（文件方式打开）
主机名:域名、IP地址
端口号:如果是http协议，默认端口号80；如果是https协议，默认端口号为443；

当跳转目标和当前页面的协议相同时可以省略协议
http://127.0.0.1:5500/2799.html
http://acm.scau.edu.cn:8000

<a href="//acm.scau.edu.cn:8000">oj实验系统</a>
  1. 相对路径书写方式(一般会有提示) 以./开头，./表示当前资源所在的目录（当前目录）...① 可以书写../表示返回上一级目录...②
(文件夹)壹
{
    （文件夹）一
    {
        1.html
    }
    2.html
    3.html

}

假设当前在2.html

<a href="./(文件夹)一/1.html">name</a>...①
<a href="./../2.html"></a>...②
