为网页添加样式

术语解释

<style>
    h1{
        color :rgb(255, 145, 0);
        background-color: darkred;
        text-align: center;
    }
</style>

CSS规则=选择器+声明块

选择器

选中元素[样式非常多]

  1. ID选择器[ id="a"->#a{} ]
  2. 元素选择器
<head>
    <style>
        /* 元素选择器 */
        h1{
            color :rgb(240, 88, 37);
            background-color: darkred;
            text-align: center;
        }
        /* ID选择器 */
        #dyht{
            color: rgb(197, 82, 43);
            background-color: rgb(139, 0, 0);
            text-align: center;
        }
    </style>
</head>
<body>
    <!-- 元素选择器 -->
    <h1>
        <br><br>大虞海棠新婚快乐<br><br><br>
    </h1>
    <!-- ID选择器 -->
    <p id="dyht" style="font-size: xx-large;font-weight: bolder;">
        <br><br>大虞海棠美帝最真<br><br><br>
    </p>
</body>
  1. 类选择器[非常灵活，需要就加上 class="a"->.a{} ]
<head>
    <style>
        .red{
            color: firebrick;
        }
        .bolder{
            font-weight: bolder;
        }
        .center{
            text-align: center;
        }
    </style>
</head>
<body>
    <h1 class="red bolder center">
        <br><br>大虞海棠暴富<br><br><br>
    </h1>
</body>

声明块

出现在大括号中，声明块中包含很多声明，每一个声明(属性)表明每种样式的规则

css代码书写的位置

  1. 内部样式表 书写在style元素中，方便提前读取css代码
  2. 内联样式表，元素样式表 直接书写在元素的style书写中
  3. 外部样式表[推荐] 将样式书写到独立的css文件中
<!-- 写在head里 -->
<link rel="stylesheet" href="./外部样式表.css">
<!-- 单独创建的css文件 即外部样式表-->
.red-big-center{
    color: crimson;
    font-size: 5em;
    text-align: center;
}
<!-- 应用 -->
<p class="red-big-center">
    大虞海棠毕业快乐
</p>

1). 外部样式可以解决多页面样式重复的问题 2). 有利于浏览器缓存，从而提高页面响应速度，大幅度提高页面响应效率 3). 有利于代码分离，html和css分开，有利于阅读和维护
